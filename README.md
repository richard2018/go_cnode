# go_cnode

#### 项目介绍
http://cnodejs.org/ 是一个用nodejs语言编写的开源论坛  
该项目是打算是用golang语言重写  
该项目部署在 http://fenghuangyu.cn:9035 账号密码admin   
#### 安装
go version 1.12.4  
用了go mod 管理依赖  

安装mysql  
安装mongodb  
安装redis   
 
## 使用

### 使用命令行

```bash
$ git clone https://github.com/dangyanglim/go_cnode.git
$ cd go_cnode
$ redis-server
$ go run main.go router.go         # 访问 http://localhost:9035
```
